## Run Locally

```bash
# 1. Clone
git clone <repo-url>
cd afredzhou

# 2. Install packages
npm install

# 3. Configure environment variables
cp .env.local.example .env.local  # add your Sanity keys here

# 4. Start the dev server
npm run dev
```

Then open:
* Portfolio ‚Üí http://localhost:3000
* Sanity Studio ‚Üí http://localhost:3000/studio
